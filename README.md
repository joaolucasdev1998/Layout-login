📌 Login Page
Este projeto é uma página de login desenvolvida com HTML e CSS, com foco em praticar a construção de interfaces web e a organização de elementos em tela.
O layout simula uma tela de autenticação simples, como as usadas em aplicações reais, reforçando conceitos de estruturação semântica, estilização com CSS e uso de Flexbox para alinhamento.
🚀 Objetivos do projeto
Praticar criação de formulários HTML
Melhorar organização de layout com CSS
Treinar Flexbox e alinhamento de elementos
Desenvolver noções de UI (interface do usuário)
🛠️ Tecnologias utilizadas
HTML5
CSS3
